# RollCall  
随机点名器  
>可以直接编辑名单，也选择文件导入名单  
>可以设置自动或者手动停止，默认是自动停止  
>自动模式可以设置时间，到了指定时间后自动停止；默认为3秒  
>默认有音效，但也可以选择静音
